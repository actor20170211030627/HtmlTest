1. javascript 简介

  1. javascript是因特网上最流行的脚本语言(脚本语言是一种轻量级的编程语言).
  2. javascript通常被直接嵌入HTML页面.
  3. javascript是一种解释性语言(代码不进行预编译).
  4. javascript被数百万计的网页用来改进设计、验证表单、检测浏览器、创建cookies，以及更多的应用。
  5. javascript被设计用来向 HTML 页面添加交互行为.
  6. 引入外部js:<script type="text/javascript" src="../js/jquery-1.8.3.js" ></script>

2. javascript 的组成部分离线手册搜索:pro_js_implement.asp


  1. ECMAScript:它是整个 javascript 的核心,包含(基本语法、变量、关键字、保留字、数据类型、语句、函数等等)
  2. DOM：文档对象模型，包含(整个 html 页面的内容)
  3. BOM：浏览器对象模型,包含(整个浏览器相关内容)
3. javascript 语法
  1. 区分大小写
  2. 变量是弱类型的(String str=”aaa” ,var str=”123”;)
  3. 每行结尾的分号可有可无(建议大家写上)
  4. 注释与 java、php 等语言相同。
4. javascript 注释
  1. 单行注释以 // 开头
  2. 多行注释以 /* 开始，以 */ 结尾。

6. javascript 数据类型

Javascript 数据类型分为原始数据类型和引用数据类型

  1. 原始数据类型：离线手册搜索:pro_js_primitivetypes.asp
    1. string
    2. number
    3. boolean
    4. null
    5. undefined
  2. 引用数据类型：
    1. Array(数组) 离线手册搜索:jsref_obj_array.asp
      1. var cars=new Array();
        cars[0]="Audi";
        cars[1]="BMW";
        cars[2]="Volvo";
        cars[100]="Test"; 注意:这时候cars.length=101
      2. var cars = new Array(size); 如果赋值的时候超过size,那么cars的长度=实际长度.如果赋值的时候没超过size,那么cars的长度=size
      3. var cars=new Array("Audi","BMW","Volvo");

        1. 连接两个或更多的数组，并返回被连接数组的一个副本。
        2. arr1.concat(arr2,arr3...);==>1,2,3,4...以逗号隔开.
      7. join(); (离线手册搜索:jsref_join.asp)
        1. 用于把数组中的所有元素放入一个字符串。元素是通过指定的分隔符进行分隔的。
        2. arr.join(); ==>George,John,Thomas
        3. arr.join("."); ==>George.John.Thomas

8. javascript 运算符

其它运算符与 java 大体一致，需要注意其等性运算符。

  1. == 它在做比较的时候会进行自动转换。
  2. === 它在做比较的时候不会进行自动转换。
  3. JavaScript里面没有equals。
9. javascript 获取元素内容
  1. 获取元素:var element = document.getElementById(“id 名称”);
    1. 如果id是一个字符串,那么必须加上引号,如果是一个变量那么不需要。
    2. ★★注意:这个方法只能在函数内赋值,不能在函数外赋值,否则无效★★
  2. 获取元素里面的值:document.getElementById(“id 名称”).value;

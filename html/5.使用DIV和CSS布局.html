1. <DIV>

Div 它是一个 html 标签，一个块级元素(单独显示一行)。它单独使用没有任何意义，必须结合 CSS 来使用。它主要用于页面的布局。

1.这是div的 元素选择器(凡是是div且没有加class或id的都会是这个样式):
<style>
 div {
  border: 1px solid red;
  font-size: 30px;
  margin-top: 2px;
  color: green;
 }
</style>
2.这是div的 类选择器(class),多个div样式相同的时候用class:
<style class="div2">
 .div2 {
  border: 2px solid green;
  font-size: 40px;
  margin-top: 2px;
  color: blue;
 }
</style>
3.这是div的 id选择器,单个div的时候用id:
<style id="div3">
 #div3 {
  border: 3px solid blue;
  font-size: 50px;
  margin-top: 2px;
  color: gold;
 }
</style>

4.这是div的 层级选择器,多个层级的时候用,示例:
<style>
 div p {
  border: 4px solid gold;
  font-size: 40px;
  margin-top: 2px;
  color: black;
 }
<style>

5. 属性选择器,示例给input设置背景颜色
<style>
 input[type='text'] {
  background-color: red;
 }
</style>

2. <SPAN>

Span 它是一个 html 标签，一个内联元素(不单独显示一行)。它单独使用没有任何意义，必须结合 CSS 来使用。它主要用于对括起来的内容进行样式的修饰。

3. <CSS>

  • Css 指层叠样式表( Cascading Style Sheets)
  • 样式定义 如何显示 HTML元素
  • 样式通常存储在 样式表 中
  • 把样式添加到 HTML 4.0 中,是为了 解决内容与表现分离的问题
  • 外部样式表可以极大提高工作效率
  • 外部样式表通常存储在 CSS文件 中
  • 多个样式定义可 层叠为一

2.1 <CSS>的作用
  1. HTML 它是整个网站的骨架.
  2. CSS 它是对整个网站骨架的内容进行美化(修饰)
2.2 <CSS>的 引入方式
  1. 内部引入: <style>直接写在html的head里面
  2. 行内引入: 直接在标签里写style="..."
  3. 外部引入: 在head里面< link rel="stylesheet" href="../css/test.css" type="text/css"/> 这儿的type="text/css"是默认属性,可以不写
    4. test.css里的内容示例:
    div {
     border: 1px solid red;
     font-size: 30px;
     margin-top: 2px;
     color: green;
    }
